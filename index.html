Instructions

1. Calculate Character Height

  1. Entweder unter "Screen" einen Bildschirm wählen oder unter "Manual entry" die Werte eingeben.
  2. Click "Calculate Character Height" button.
  3. You'll see the result in the "Minimum character height" section

2. Add a new Screen

  1. Eingabefelder unter "Neuen Bildschirm anlegen" ausfüllen.
  2. Auf den Button "Bildschirm speichern" klicken.
  3. Den Text aus dem eingeblendeten Textfeld kopieren und in .misc/bildschirme.js unter dem richten Bildschirmtyp (1-Monitor, 2-Laptop, 3-Tablet, 4-Smartphone) einfügen.
  4. Datei speichern.
